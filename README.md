# Motel Pricings - (Ineteractive Data Using Tableu)
## Project Objective 
A Motel Owner wants to know where and when to open a Motel and all the pricings for them to grow their sales in the upcoming years.

## Data Set Used
- <a href="https://github.com/thechampionjason-stack/My-Tableu-Project/blob/main/AirBnb%20Full%20Project.twbx">Data set</a>

## Questions
- Compare Sales in different Zipcodes using a map.
- Which month has the highest sales?
- Which time of the Year has the highest Sales?
- Which bedroom sizes cost the most?
- The revenue of over the years.
